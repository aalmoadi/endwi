This folder contains independent validation data for ENDWI method.
## Independent Validation Case

To assess the transferability and robustness of the ENDWI method, an independent flood event from a previously published study was utilized.

The case study corresponds to the flood event reported in:

"TLE-FEDformer: A Frequency-Domain Transformer Framework for Multi-Sensor Multi-Temporal Flood Inundation Mapping"

DOI: doi.org/10.3390/rs18060895
