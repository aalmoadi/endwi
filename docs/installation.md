# Installation

## Stable release

To install ENDWI, run this command in your terminal:

```
pip install endwi
```

This is the preferred method to install ENDWI, as it will always install the most recent stable release.

If you don't have [pip](https://pip.pypa.io) installed, this [Python installation guide](http://docs.python-guide.org/en/latest/starting/installation/) can guide you through the process.

## From sources

To install ENDWI from sources, run this command in your terminal:

```
pip install git+https://github.com/aalmoadi/endwi
```
